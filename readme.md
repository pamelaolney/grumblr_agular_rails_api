# Welcome to the API for Grumblr

Once setup, you can access your grumblr api at `http://locahost:3000`.

## Setup

1. Install dependencies, setup db

  ```bash
  $ cd grumblr_rails_api
  $ bundle install
  $ rake db:setup
  ```

2. Start your server

  ```sh
  $ rails s
  ```

3. Your api is now available at `http://localhost:3000`
